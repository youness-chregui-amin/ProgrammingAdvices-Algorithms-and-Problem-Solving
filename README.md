#  About the Programming Roadmap

The **ProgrammingAdvices Roadmap** is a comprehensive programming learning path created by **Dr. Mohammad Abu-Hadhoud**.

It guides students step by step from the fundamentals of programming to advanced software engineering topics, including:

* Programming Foundations
* Algorithms & Problem Solving
* C++
* Object-Oriented Programming (OOP)
* Data Structures
* Databases
* Design Patterns
* Software Engineering
* Desktop Development
* Web Development
* And many other advanced topics.

The solutions in this repository cover only the **Algorithms & Problem Solving** section of the roadmap.
